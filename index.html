ReMoSS

Remote Mobile Surveillance System is an client/server app that can transform your Android device into a surveillance camera. You can use multiple devices to connect to the server and use it as a portable surveillance system.

Configuration instructions

  1. Download and install the Server App on your computer (download links below)
  2. Download the Client App on your Android device (get it on the Google Play Store here)
  3. Run the Server App on your computer
  4. Run the Client App on your Android device and connect to the Server App with the conection string displayed on the Server App

Authors and Contributors

Created by Frédéric Begin (@fredbegin11) and Renaud Lainé (@Elrhino).

Additional Notes

  • Make sure your Android Device is on the same network/subnet than the computer
  • AndroidCamera uses TCP port 44444 and UDP 40000 to 40006
